# 👟 Vue Sneakers — Modern E-commerce Application

[![Vue.js](https://img.shields.io/badge/Vue.js-3.x-4fc08d?style=for-the-badge&logo=vue.js)](https://vuejs.org/)
[![TailwindCSS](https://img.shields.io/badge/Tailwind-CSS-38B2AC?style=for-the-badge&logo=tailwind-css)](https://tailwindcss.com/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](https://opensource.org/licenses/MIT)

**Vue Sneakers** is a fully functional e-commerce storefront designed for seamless shopping experiences. The application features dynamic data fetching, complex state management, and a polished UI with smooth interactive elements.

## ✨ Key Functionality

- **Advanced Cart System:** Interactive shopping cart with real-time price updates and persistent state.
- **Favorites & Wishlist:** Ability to save products to a personalized favorites list.
- **Dynamic Content:** Real-time product searching, filtering, and sorting by various parameters.
- **Checkout Flow:** Integrated order placement system with backend data synchronization.
- **Premium UX:** Fluid list transitions and UI feedback powered by AutoAnimate.
- **Adaptive Layout:** Mobile-first responsive design built with Tailwind CSS.

## 🛠 Tech Stack & Architecture

- **Core Framework:** Vue 3 using **Composition API** for scalable logic.
- **Navigation:** Vue Router for seamless SPA transitions.
- **Backend Integration:** REST API communication via **Axios**.
- **Data Source:** Remote JSON-backend (Mokky.dev).
- **Styling:** Utility-first CSS with Tailwind.
- **Developer Experience:** Vite, ESLint, Prettier.

## 🚀 Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/heysabyrka/vue-sneakers.git
   ```

2. Install project dependencies:
   ```bash
   npm install
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
